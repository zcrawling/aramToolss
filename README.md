# KHU NOTICE SERVER  
경희대학교의 각 게시판에 올라온 공지사항들을 정기적으로 크롤링하여 보기쉽게 정리하는 웹사이트입니다.  
### STATUS: 개발중(예상기한: 01/25 ~01/30)  
  
  
이 프로젝트는 CrowCpp(https://github.com/CrowCpp/Crow/tree/master#)를 사용합니다.   
  
## DEPENDENCY  
C++ 20  
CMAKE 4.1+    
libasio-dev (1.28+)  
libcurl4-openssl-dev (8.5.0+)  
python3 (3.11+)

### 개발일지  
  
1: 개발의도     
경희대의 각 사이트들에 분산되어 올라오는 공지사항을 크롤링하여, 하나의 단일 웹사이트에서 볼 수 있도록 만드는 것.    
    
2: (01/25) C로 짜인 몽구스라는 웹 라이브러리를 이용하여 크롤러와 서버, 데이터베이스를 단일 C++ 프로그램안에서 구현할 것이다.  
프로젝트에 알맞는 구조의 database를 작성하였다.  
    
3: (01/26) 몽구스로 인해 Cstyle코드와 C++style코드가 난잡하게 얽혀있다. 도저히 메모리 안정성을 고려할 수 없다.     
    
4: (01/27) 몽구스를 완전히 deprecate하였다. 대신 서버는 Crow, 크롤러는 Python3로 대체한다.    
프로세스를 두개로 분리하니 프로세스간의 통신 및 멀티스레딩 프로그래밍이 필수적이다.    
통신은 유닉스 도메인 소켓, 멀티스레드는 STL인 <thread> 및 <mutex>를 이용하여 구현할 것이다.    
    
5: (01/28) database를 thread-safe하게 수정하고, 크롤러 프로그램과 통신을 구현하였다.  
  
6: (01/29) 크롤러를 구현하고 일부 코드를 수정하였다.
